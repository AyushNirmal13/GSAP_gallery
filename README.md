# ScrollCanvas 🎬

*A GSAP-powered animated gallery experience*

## 🚀 Overview

ScrollCanvas is an interactive web gallery built using **GSAP (GreenSock Animation Platform)**.
It focuses on **smooth scroll-based animations, immersive UI, and modern frontend design**.

The project demonstrates how motion can enhance storytelling and user experience in web interfaces.

## 🎯 Features

* 🎞️ Scroll-based animations using GSAP
* ⚡ Smooth and high-performance transitions
* 🧭 Interactive gallery layout
* 🎨 Modern UI with creative motion effects
* 📱 Responsive design

## 💡 Key Concept

GSAP is a powerful animation library that enables high-performance animations across browsers and devices. ([Awwwards][1])

This project uses GSAP to create:

* Scroll-triggered animations
* Parallax-like effects
* Smooth visual storytelling
